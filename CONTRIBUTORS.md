## Как добавить себя как контрибьютора на GitHub

Чтобы ваш аватар отображался в разделе Contributors на GitHub, нужно:

1. Убедиться, что email в ваших Git настройках совпадает с email в GitHub аккаунте
2. Сделать хотя бы один коммит с этим email

### Проверка текущего email:
```bash
git config user.email
```

### Установка email для Git:
```bash
git config user.email "your-email@example.com"
git config user.name "Your Name"
```

### Или для всех проектов:
```bash
git config --global user.email "your-email@example.com"
git config --global user.name "Your Name"
```

После этого сделайте любой коммит (например, исправление опечатки в README), и ваш аватар появится в Contributors!
